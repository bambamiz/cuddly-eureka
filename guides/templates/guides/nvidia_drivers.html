Nvidia drivers

I'm gonna go over how to download your GPU drivers, uninstall your current drivers using DDU(a tool to uninstall drivers), and install your new drivers

note:using DDU to is optional but recommendedd, I'll denote the with a blue background.

At the bottom of the page there is a short explanation on why I recommend updating your drivers.

1. Download your GPU drivers.

  • Open up your browser and go to Nvidia's website Nvidia drivers
  • You'll notice a few dropdown menus, select the option that matches your GPU and click the search button.

    • note:if you're unsure which GPU you have *click here*.

1. Continued

  • The search results will return the driver for your system, click the green download button to save it to your system.

    • note:I've included an image below(fig 2), but keep in mind that the version will change as they continue to release drivers.

(optional)

2.1 Download DDU, restart in safemode and uninstall your current drivers.

  • DDU Download DDU, the version number might be different. e.g image(fig 3)
  • Now scroll all the way down to the bottom and you will see a download section. e.g (fig 4)
  • You'll see a few option, select the download link thats from your region. The link will open up a new page where the download will start automatically.

2.2 unzip DDU

  • When you download DDU you'll notice it's a zip file, you'll need a tool to unzip it. If you don't already have one*insert link* i recommend. note: I will be using 7-zip for this guide.
  • Go to your file explorer (WIN key + e) and go to your downloads folder. You'll see the DDU folder you just downloaded, right click it to pull up a sub menu.
  • On the sub menu you'll want to look for 7-zip then hover over it to pull up yet another sub menu where you'll find the option to open archive, click it. e.g (fig 5)
  • Extract the file ending in exe to your downloads by highlighting it and clicking copy on the top menu. Next you'll see a new window confirming the location where you'll be copying to. e.g (fig 6).

    • note: the path should be C:\Users\username\Downloads\ the example below has bamba becuase that is my username.

  • You'll see the new file in your downloads. Like before you're gonna right click it, hover over 7-zip, and open archive. e.g (fig 7)
  • In the new window you'll only have one option, highlight it and click copy. A window confirming the location will apear, click ok. e.g (fig 8)

2.3 restart in safemode

  • Next you will restart in safemode and turn off internet access (probably a good idea to open this section on another device) to prevent windows from automatically installing drivers you don't want.
  • On your keyboard press WIN+r keys to open the run menu. In the text box type msconfig and click OK. e.g (fig 9)
  • You will now be on the system configuration window where you will see 5 tabs on top, click the Boot tab.
  • On the boot options check Safe boot and Minimal then click OK. e.g fig(10)
  • Lastly you will get a window reminding you to save changes before you restart. Restart whenever you're ready. e.g (fig 11)

2.4 uninstall your current drivers

  • You shuld now be in safemode (confirmed by the text on the corners of your screen.) Next you will use DDU to remove your current drivers.
  • In your downloads folder you'll see a unzipped folder named DDU, open it and find DDU Display Driver Unintaller, launch it. e.g (fig 12)
  • After you launch it you'll see a message about this being the firstime you've used DDU and a options menu. In the options menu you'll see an Nvidia section, make sure they are all selected. e.g (fig 13)
  • On the DDU window you just opened you'll see two drop down menus, on the top one select GPU, on the bottom one select Nvidia and select Clean and restart. e.g (fig 14)
  • When your pc restarts it might look zoomed in or pixelated this is normal.

3. Restart in safemode

  • Next you will restart in safemode and turn off internet access (probably a good idea to open this section on another device) to prevent windows from automatically installing drivers you don't want.
  • On your keyboard press WIN+r keys to open the run menu. In the text box type msconfig and click OK. e.g (fig 15)
  • You will now be on the system configuration window where you will see 5 tabs on top, click the Boot tab.
  • On the boot options check Safe boot and Minimal then click OK. e.g fig(16)
  • Lastly you will get a window reminding you to save changes before you restart. Restart whenever you're ready. e.g (fig 17)

4. Install your drivers

  • Go to your file explorer (WIN key + e) and go to your downloads folder. Find the Nvidia driver you just downloaded and double click it. You'll notice windows defender pop up go ahead and click run. e.g (fig 18)
  • You'll see a window confirming where you would like to save the driver files. Leave it set to it's default location and click OK. When it is done, the Nvidia installation window will open automatically. e.g (fig 19)
  • The Nvidia installation window will prompt you with their user aggrement, aggree and continue whenever you're ready. e.g (fig 20)
  • Next you'll be on installation options, select custom and click next. e.g (fig 21)
  • Now you'll be on custom installation options, select Graphics Driver and HD Audio Driver (audio driver is not necessary, but if you're unsure if you need it go ahead and install it). e.g (fig 22)
  • Nvidia installer has finished, here you'll have two options Restart now or Restart later, select Restart later. Before you restart you wanna set your pc to boot up normally (you are still set to boot in safemode). e.g (fig 23)
  • You wanna redo steps from earlier, press WIN+r keys, type msconfig, in the general tab make sure to select Normal startup and click OK. On the confirmation window click restart. e.g (fig 18)

Awesome, you've now have the latest drivers installed.
